# nvim

Personal Neovim configuration built on LazyVim.

## Highlights
- Lazy-loaded plugin setup
- Custom keymaps, options, and autocmds
- Theme overrides and UI tweaks

## Structure
- `init.lua`: entry point
- `lua/config/`: core config (options, keymaps, autocmds)
- `lua/plugins/`: plugin specs and overrides
- `plugin/after/`: runtime tweaks

## Notes
- `lua/plugins/copilot-chat-dev.lua` is intentionally ignored.
